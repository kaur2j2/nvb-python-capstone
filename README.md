# nvb-python-capstone
NVB Python Summer 2026 Capstone Project


What does it do?

This project analyzes a biomedical patient outcomes dataset using Python and pandas. The goal is to identify which diagnosis and age group have the highest 30-day hospital readmission rates. The program loads the dataset, checks for missing values, calculates readmission rates, creates two bar graphs, and summarizes the results.


How to run it:

Open the notebook in Google Colab.
Upload the nvb_patient_outcomes.csv dataset when prompted.
Run each section of the notebook from top to bottom.


Skills demonstrated:

his project demonstrates data analysis with pandas, function creation, data grouping, missing value checks, and graphing with matplotlib.


Key findings:

Cardiovascular Disease had the highest 30-day readmission rate, and the 45–59 age group had the highest readmission rate in the dataset.
